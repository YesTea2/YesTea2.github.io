In Remainder Matters, players navigate a series of obstacles to push numbered bricks into the correct position. The goal of the game is to arrange the numbered bricks so that their values match the target number on the board. This challenging and engaging game combines elements of math and logic to provide a unique and enjoyable experience.


During the development of this project, I sought to create a system for level creation that was flexible and efficient. To achieve this, I used an array of transforms with a numbered grid on paper. This allowed me to easily specify where specific tiles should be spawned by passing in the corresponding numbers through one of three functions. By using this approach, I was able to avoid the need for separate scenes or prefabs in the inspector. Instead, levels are generated dynamically by simply wiping the playing area and spawning new bricks as needed. This approach allows for greater flexibility and easier level creation.

A Rift Between

A Rift Between was a project in which the player assumed the role of a spaceman searching for resources to repair their broken ship. The game was presented in a 2.5D style, featuring a unique mechanic in which the player controlled their robot companion using a command wheel to assist in combat, resource gathering, and defense against incoming attacks.


This project provided a valuable learning experience, as I developed skills in inventory management, mission design, shader creation, and data saving and loading. I also documented much of my learning process through developer logs on YouTube, which serve as a useful reference to reflect on my growth and progress as a developer.


Though I have continued to learn and evolve since then, it is satisfying to be able to look back and see where I was at a particular point in my journey.

Puzzle Jump is a casual puzzle game that allows players to create and platform through their own levels. In each level, players are given a selection of platforms to use to reach all of the objectives.


I implemented a system using dictionaries that contain arrays of integers to use as a index to add and alter the platforms available per level. This allowed for dynamic level creation and easy scalability.

The Liquid Dream Machine is a game that challenges players to collect as much fluid as possible by strategically arranging platforms and objects to guide the fluid over obstacles and into a container.


I placed a strong emphasis on developing a user-friendly interface and creating a webGL build. Additionally, I utilized the animator feature in Unity to create engaging text effects for the first time.
