Projects

Puzzle Jump : A Platforming Adventure

A 2D Build Your Own Platformer
This project was an entry for the Thinky Games Game Jam.
It features two game modes: a platforming editor and a platforming mode.
In the editor, the player is given a set of platforms to use to build a level. The objective in platforming mode is to clear all objectives for each level.
The project focuses on easily scalable systems to enable fast expansion with additional platforms and levels.

MTG Interactive Twitch Overlay

A Interactive Overlay for twitch streamers
This is a transparent overlay adding interactivity to a streamers Twitch chat.
It features two channel point redemptions that use the SkryFall public API
One redemption displays a random Commander MTG card to the viewer, along with an audio clip.
The other allows the viewer to search for a magic card and display it to all current viewers and the streamer.
The application also includes a greeting message for each chatter and a chatbox that allows the streamer to see chat without a second monitor.

Remainder Matters

A Sokoban style game about math
The goal of this project is for the player to push numbered bricks into the play area and match the number on the brick with the number on the side of the play area.
Each push the player gives to the brick reduces its number by one. Additionally, there are operand bricks that can perform arithmetic operations on the current brick when pushed next to it.
A major focus during development was creating a system that enables rapid level creation using code instead of manually configuring each level in the Unity inspector.
This project also features a save system and character selection.
